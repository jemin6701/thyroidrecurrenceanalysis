# thyroidrecurrenceanalysis
A machine learning project predicting whether a thyroid cancer patient is likely to experience recurrence, using clinicopathologic data from the UCI Machine Learning Repository. I performed in-depth EDA across 13 clinical features — including tumor stage, lymph node involvement, pathology type, and treatment history — then built and compared six classification models. The final tuned Random Forest model achieved 98.7% accuracy with 100% precision in identifying recurrence cases. Tools: Python (Pandas, Scikit-learn, Seaborn, Plotly), One-Hot Encoding, Power BI, Joblib (model deployment)
